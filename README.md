# OpticaLab

## What is this?

It's a project for a class at the University of Waterloo; part of my undergrad.

## Tests

I've attached some modular test scripts and configs under `tests/`. The easiest way to run these is (from the root directory),
```python3 test_suite.py```

If you plan on running any of these tests individually, you can also run commands like,
```python3 tests/test_XX.py config_file.txt```

where you have filled out `config_file.txt` with the various configuration options.
